The bar chart displays the top 10 most common bacteria cultures found in the belly button of the selected test subject. The y-axis displays the OTU ID of each bacteria culture, while the x-axis shows the number of samples found for that specific culture.

The gauge chart displays the number of times per week the test subject reported washing their belly button

The bubble chart displays all of the bacteria cultures found in the belly button of the selected test subject. The OTU ID of each bacteria is plotted on the x-axis, while the number of that specific culture is plotted on the y-axis. Each circle size is also based on the sample values for each bacteria culture and the culture names can be viewed by hovering over the corresponding circle.
